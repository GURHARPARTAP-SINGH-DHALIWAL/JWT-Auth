# [JWT Authentication](https://jwt-auth-gsd.herokuapp.com/)

A project for learning authentication using JWT

## Technologies Used

* Node.js

* bcrypt

* cookie-parser

* dotenv

* ejs

* express

* express-ejs-layouts 

* validator

* jsonwebtoken

* mongoose

* nodemon 

* MongoDB

* Materialize CSS

* CSS


## Run Locally


Clone the repo or download zip and run the following commands-

`npm install`

`npm start`
